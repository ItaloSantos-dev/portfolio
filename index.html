Valorize

Uma API que usa API´s públicas do governo para calcular rendimento de um investimento, aidna possibilitando escolher a moeda de entrada e a moeda de saída.

O que soluciona

Esta é uma simples API mas que com um front end poderia facilmente ser microsaas, a ideia é poupar tempo do usuário, imagine que ele planeja fazer uma viagem para o exterior e quer saber o quanto de investimento em dólar no Brasil ele tem que fazer para conseguir a quantidade necessária para levar na viagem, a valorize seria uma mão na roda para esse usuário pois manualmente ele teria que, buscar cotação das moedas escolhidas, pegar a cotação correta para conversão, saber a taxa atual do investimento escolhido e calcular usando a fórmula de juros composto, todo esse trabalho a valorize poupa para o usuário. ps. Atualmente a aplicação está instável pois as API´s públicas do governo estão com problemas.

Conhecimentos aplicados

POO, MVC, layered architecture, SQL, migrations, dockerização da aplicação, consumo de API´s externas, Lógica pura para cálculo de juros compostos
